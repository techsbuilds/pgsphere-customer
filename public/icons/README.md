# PWA Icons for Pgsphere

This directory contains the app icons for the Pgsphere PWA application.

## Required Icon Sizes:
- icon-16x16.png
- icon-32x32.png
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Icon Source:
The icons should be created from the Pgsphere logo provided by the user.

## Usage:
- **Favicon**: 16x16, 32x32
- **Apple Touch Icon**: 192x192
- **Android**: 72x72, 96x96, 128x128, 144x144, 152x152, 192x192
- **Splash/Large**: 384x384, 512x512

## Format:
All icons should be PNG format with transparent or white background.
